# homebrew-muchq

Homebrew tap for [muchq](https://github.com/muchq) projects.

## Usage

```
brew tap muchq/muchq
brew install impact-mcp
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [impact-mcp](Formula/impact-mcp.rb) | Local-first AI agent helping engineers amplify impact and grow in their role |
